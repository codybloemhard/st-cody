# st-cody
My Suckless ST fork.
## Extra Features
- Translucent background
- Scrollback per page
- Colours can be set with cli arguments
- Colours can be set with Xresources
## Patches aplied
- st-alpha-0.8.2.diff
- st-scrollback-20200419-72e3f6c.diff
## Some Shortcuts
### (Meant for QGMLWY layout)
C(Control), S(Shift)
- C-S c = copy
- C-S v = paste
- C i = scroll up a page
- C k = scroll down a page
