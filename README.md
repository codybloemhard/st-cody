## Significant Shortcuts
C(Control), S(Shift)
- C-S c = copy
- C-S v = paste
- S up/down = scroll n lines up/down
- S pgup/pgdn = scroll n lines up/down
- C pgup/pgdn = scroll page up/down
- C-S pgup/pgdn = increase/decrease fontsize
